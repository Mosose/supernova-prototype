# 🏛️ The Supernova Synthesizer
**A $0 Prototype for Lossless Democracy**

### 💡 The Concept
Inspired by Dean Dr. Henry Abramson's lectures on AI and historical context, this tool uses Large Language Models and Clustering algorithms to turn thousands of conflicting public opinions into a clear, actionable "Democratic Signal."

### 🛠️ How it Works
1. **Integrity Filter:** Removes bot attacks and duplicate spam.
2. **Semantic Mapping:** Uses `all-MiniLM-L6-v2` to map opinions into a 128-dimensional vector space.
3. **Synthesis:** Clusters ideas into "The Consensus," "The Friction," and "The Supernova Insight" (innovative outliers).
